### 🌟 Project Overview
**OmniSolver** is a specialized computational project focused on finding high-precision solutions for a 5x5 complex system of linear equations. By implementing and comparing four distinct mathematical strategies, this engine provides a benchmark for efficiency, convergence speed, and numerical stability.

---

### 🚀 Implemented Numerical Methods
The engine solves the system using two primary approaches:

#### A. Iterative Methods (Convergence Based)
* **Jacobi Method:** A recursive algorithm that computes each variable independently based on previous state values.
* **Gauss-Seidel Method:** An optimized iterative process that utilizes the most recently updated values within the same iteration for faster convergence.

#### B. Direct Methods (Exact Solutions)
* **Cramer's Rule:** Utilizing determinants to solve for each variable directly.
* **Matrix Method:** Implementing matrix inversion ($A^{-1} \cdot B$) for a structural mathematical solution.

---

### 📊 Performance & Comparative Analysis
The project includes a detailed performance log:
* **Convergence Tracking:** Comparison between Jacobi and Gauss-Seidel iteration counts (e.g., 29 vs 17 iterations).
* **Accuracy Verification:** Cross-validating results across all four methods to ensure floating-point precision.
